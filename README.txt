Este script tem a função de verificar de minuto em minuto a cotação do Bitcoin e Litecoin com base no MercadoBitcoin (www.mercadobitcoin.com.br).

Funcionalidades:
	- Exibir cotações de minuto em minuto para o usuário;
	- Aviso por email quando a moeda atinge o valor determinado pelo usuário para compra;
